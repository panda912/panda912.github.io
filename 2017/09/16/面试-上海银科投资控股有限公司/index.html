发表于 2017-09-16

笔试:

  1. 选择题 (一看就是网上荡的)
  2. 看代码写输出结果, 主要考 Java 基础(list拷贝, 继承, String 和 StringBuilder)
  3. MVC 原理, MVC 在 android 中的应用
  4. requestLayout invalidate postInvalidate 的区别
  5. Handler 原理, 子线程创建 Handler 需要注意什么?
  6. sleep 和 wait 的区别

一面:

  1. activity 四种启动模式; ABCD 四个页面分别对应 standard singleTop, singleTask, singleInstance四种启动模式, A->B->C-D, 分析 Task 栈, 再 D->A->B->C, 分析 Task 栈
  2. fragment 生命周期; fragment 如何实现懒加载
  3. 有没有做过嵌套滚动的页面? 如何实现的? CoordinatorLayout 原理
  4. MVP
  5. js 和 native 通信
  6. 4.2以下的 webview 漏洞, 怎么处理 (不准使用第三方)
  7. 目前项目中使用的 http协议 是哪个版本? http1.1 和 http1.0 有什么区别
  8. 介绍一下 RxJava 的特点

二面:

  1. 为什么会页面卡顿; 哪些因素会导致页面卡顿; 如何排查页面卡顿; 如何优化页面卡顿; 频繁 GC 会不会导致页面卡顿, 为什么
  2. 为什么会内存泄露; 泄露的是栈内存还是堆内存; 平时使用什么工具分析内存泄露, 原理是什么
  3. Stack Overflow(栈溢出), 为什么栈会溢出
  4. android 手机的堆内存有多大
  5. 如何优化布局文件
  6. 使用帧动画导致内存升高, 在使用帧动画的前提下如何优化内存; (PS: 压缩图片质量真的能够减少内存占用吗)
  7. 在屏幕左下角垂直绘制ABC3个文字; 绘制圆环
  8. http status code 304
  9. 启动页广告图, 固定的图片链接, 客户端如何知道广告图片已经更新了
  10. 如何优化 app 启动速度; 多进程会不会影响 app 启动速度
  11. 多线程数据同步方式; 用过哪几种
  12. 锁(lock)有没有用过; lock 和 wait 的区别; 如何释放锁, 并通知其他线程当前可用
  13. 两个接口都请求成功之后才做下面的操作, 用 rxjava 怎么实现
  14. 有没有使用过 socket? 使用 socket 发送消息, 使用 Queue 管理消息队列(先进先出), 如果突然发送一个优先级高的消息, 如何优先处理这个优先级高的消息
  15. 平时遇到问题都是怎么解决的? 有没有 github 账号? 有没有成熟的开源项目?
  16. 如果遇到问题, 你会怎么处理(给你一天时间, 到中午了还没有解决)
  17. 如果跨部门沟通出现问题, 你会怎么做
  18. 如果突然出现某个问题, 不知道是终端还是前端还是后台的问题, 你会怎么做
  19. 如果老大临时把你调到其他项目组, 你会怎么做, 如果那边人员不配合你, 你会怎么做
  20. 如果老大让你调研一个新技术, 你会怎么做
